Работа с одномерными массивами

1. Для значений I=1, 2,..,N вычислить число сочетаний из N по
   I и занести результаты в массив  С=(С1,С2,..,СN), используя
   Ci= n!/(i!(n - i)!).  Отсортировать  методом
   перестановок по возрастанию. На печать вывести массив С до 
   сортировки  и после нее. Здесь числo N > 0 ввести с клавиатуры.
2. Вычислить   и   записать  в  массив  значения   функции
   y=a*x**2+b*x+c при изменении x от хнач до хкон  с  шагом  h.
   Массив значений отсортировать по убыванию методом пузырька и
   напечатать.  a =2.14;  b= -4.21; c= 3.25;
   хнач = -4.5,   хкон= -33.5, h= 0.5;  напечатать результат.
3. Вывести  на  экран  сначала все  положительные  элементы
   массива сформированного с помощью случайных чисел, затем все
   отрицательные.
