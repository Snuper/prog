Построение СДП
Выполняется путем добавления вершин в уже существующее дерево.
Если дерево пустое, то добавляем новую вершину, распределив память и занеся в нее данные. В противном случае добавляем вершину к левому или правому поддереву в зависимости от соотношения добавляемого ключа с текущей вершиной.

Pas: TYPE pVertex = ^tVertex; Var p : ^pVertex; C: Vertex **p Добавление(D - данные;Root - корень); p:=@Root; DO(p^<>nil) IF (Dp^^.Data) p:=@(p^^.right); ELSE OD OD IF (p^.right=NIL) new(p); p^^.Data:=D; p^^.Left:=NIL; p^^.right:=NIL; FI Хотя этот алгоритм предназначен для построения дерева, его можно использовать для сортировки, но необходимо предусмотреть возможность включения в дерево элементов с одинаковым ключом.

В худшем случае может выродится в список.
