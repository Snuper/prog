Деревья поиска
Двоичное дерево часто используется для представления множества данных, среди которых идет поиск элементов по уникальному ключу.
Будем считать:
  1. Часть данных, хранящихся в каждой вершине, является ключом для поиска.
  2. Для всех ключей определены операции сравнения.
  3. В дереве нет вершин с одинаковым ключом.
Двоичное дерево называется деревом поиска, если ключ в каждой его вершине больше ключей в левом поддереве и меньше ключей в правом поддереве.
В дереве поиска обход слева направо дает упорядоченную последовательность.
