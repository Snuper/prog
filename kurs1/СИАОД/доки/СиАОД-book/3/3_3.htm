Индексация через массив указателей
В индексный массив вместо номеров элементов записываются их адреса.
Построение индексного массива:
  1. В массив B записываются все адреса элементов массива A. B:=(@A1,@A2,...,@An)
  2. Производится любая сортировка. При сравнении элементов массив A адресуется через массив B.
  3. Перестановки только в массиве B.

Преимущество:
Исходные данные могут располагаться не только в массиве, но и произвольно в памяти.
