Некоторые свойства деревьев
Свойство 1:
Максимальное число вершин в двоичном дереве высоты h вычисляется по формуле Nmax(h)=2^(h-1) Доказательство:
  1. уровень - 1 - 2^0
  2. уровень - 2 - 2^1
  3. уровень - 4 - 2^2
    ....
  4. уровень - 2^(h-1)
Свойство 2:
Минимально возможная высота двоичного дерева с n вершинами вычисляется по формуле hmin(n) = log(n+1) Двоичное дерево называется идеально сбалансированным (ИСД), если для каждой его вершины размер левого и правого поддеревьев отличаются не более, чем на единицу.

Свойство 3:
Высота ИСД с n вершинами минимальна и равняется [log(n+1)].
