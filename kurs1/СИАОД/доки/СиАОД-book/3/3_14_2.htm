Приближенные алгоритмы построения ДОП
Идея алгоритмов:
A1 : В качестве корня возьмем вершину с максимальным весом, аналогично будем поступать для каждого поддерева.
A2 : В качестве корня возьмем такую вершину, для которой разность весов левого и правого поддерева будет минимальна.

Алгоритм A1

Root:=nil; use:=true; DO(i:=1,...,n) v[i].use:=false; OD DO(i:=1,...,n) max:=0; index:=0; DO(j:=1,...,n) IF ((V[i].W>max)and(V[j].use=false)) max:=V[j].W; index:=j; FI OD V[index].use:=true; Добавить(Root,V[index]); OD Алгоритм A2(L,R)

wes:=0; summa:=0; IF(L<=R) DO(i:=L,...,R) wes:=wes+W[i] OD DO(i:=L,...,R) IF ((summa=wes/2)) OD FI summa:=summa+W[i]; Добавить(Root,V[i]); A2(L,i-1); A2(i+1,R); FI A1 - плохое, дерево становится равносильно случайному при стремлении n к бесконечности. Дерево A2 ассимптотически приближается к оптимальному. ИСДП не считается ДОП.
