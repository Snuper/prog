Адреса и указатели
Адрес - целочисленное 16-ое беззнаковое значение, с которым можно оперировать как с целым числом.
Для удобства работы с адресами была придумана специальная переменная - указатель, т.е. переменная, значением которой является адрес объекта конкретного типа. Значение указателя может быть не присвоенно никакому адресу, в этом случае указатель прнимает "никакое" значение - NULL,NIL. Указатель необходимо описывать, т.е. обозначить на объект какого типа этот указатель ссылается.

Pas : k : ^char; t : ^integer; C : char *k; int *t; Доступ по адресу, хранящемуся в этих указателях

Pas : значение по адресу k^,t^ C : значение по адресу *k,*t
