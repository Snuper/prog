Удаление вершины с ключом X из СДП
Найдем вершину с ключом X.
Возможны случаи:
  1. Удаляемая вершина имеет не больше одного поддерева
  2. Вершина имеет два поддерева, дерево нужно перестроить:
    На место удаляемой вершины ставится максимальная вершина из его левого поддерева, т.е. самая правая вершина его левого поддерева.

Алгоритм:

Удаление(X - ключ;Root - корень); p:=@Root; <поиск> IF (p^<>NIL) q:=p^; IF (q^.Left=NIL) p^:=q^.right ELSE IF (q^.right=NIL) p^:=q^.left ELSE {2 поддерева} ELSE{2 поддерева} r:=q^.left; s:=q; DO(r^.right<>NIL) S:=r; r:=r^.right; OD S^.right:=r^.left; r^.left:=q^.left; r^.right:=q^.right; p^:=r; FI Dispose(q);
