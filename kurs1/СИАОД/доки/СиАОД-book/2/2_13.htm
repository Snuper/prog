Метод Хоара
Был разработан Koare в 1962 году.
Общая идея:
Возьмем произвольный элемент и обозначим его за X. Просматривая массив слева, будем искать элемент A[i]>=X,просматривая массив справа, будем искать элемент A[j]<=X.Поменяем местами A[i] и A[j]. Продолжаем просматривать массив слева и справа, пока i не станет больше j, разделим массив на две части и применим тот же метод к каждой части, будем повторять до тех пор пока в каждой из них не останется по 1 элементу.

В качестве элемента X будем брать 1-ый эдемент массива A[L].

QuickSort(L,R) X:=A[L]; i:=L; j:=R; DO(i<=j) DO(A[i]<X) i:=i+1; OD DO(A[j]>X) j:=j-1; OD IF (i<=j) A[i]<->A[j]; i:=i+1; j:=j-1; FI OD IF (L<j) QuickSort(L,j); IF (i<R) QuickSort(i,R); В головной программе вызываем QuickSort с параметрами 1,n.

Проблемма глубины рекурсии.
В этом алгоритме возможно переполнение стека, чтобы этого избежать можно применить следующий алгоритм:

DO(L<R) <разделение массива> IF (j-L<R-i) QuickSort(L,j); L:=i; ELSE QuickSort(i,R); R:=j; FI OD
