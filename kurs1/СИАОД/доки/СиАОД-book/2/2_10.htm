Двоичный поиск
A = (a1,a2,...,an) a1<=a2<=...<=an Пусть задан ключ поиска X и необходимо:
1. Найти элемент с заданым ключом X
2. Найти все элементы с заданым ключом X
Замечание: если массив не отсортирован, то необходимо просмотреть все элементы.
Общая идея:
Берем средний элемент массива и сравниваем его с ключом X
1. средний элемент равен X => завершаем поиск
2. средний элемент меньше X => продолжаем поиск в правой части массива
3. средний элемент больше X => продолжаем поиск в левой части массива

Обозначим через L и R правую и левую границы массива, введем логическую переменную find(найден)

Алгоритм:

L:=1; R:=n; find:=false; DO(L<=R) m:=(L+R) div 2; IF (a[m]=x) find:=true; OD FI IF (a[m]<x) L:=m+1 ELSE R:=m-1; FI OD Недостатки:
1. На каждой итерации цикла 2 сравнения
2. Эта версия находит один элемент из множества элементов с заданым ключом=>надо просматривать и слева и справа

Алгоритм:

L:=1; R:=n; DO(L<R) m:=(L+R) div 2; IF (a[m]<X) L:=m+1 ELSE R:=m FI OD IF (a[R]=X) find:=true ELSE find:=false; Тродоемкость:
O(log2n)
