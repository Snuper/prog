Лабораторная №1
Дан массив A=(a1,...,an) из n целых чисел, n - константа.
  1. На Pascal-е и C разработать 5 процедур
    1. заполняет массив возрастающими числами
    2. заполняет массив убывающими числами
    3. заполняет массив случайными числами
    4. функция, считающая контрольную сумму элементов массива.
    5. функция, определяющая количество серий в массиве.
  2. Определить наибольший размер массива экспериментально.
  3. Экспериментально определить среднюю длину серии в массиве.
  4. Массив передавать в качестве параметра подпрограмме.
  5. * Организовать многофайловый проект, все подпрограммы в одном файле, а основная в другом.
