Основные теоремы побуквенного кодирования
Теорема Крафта.
Для того чтобы существовал префиксный код с длинами L1,...,Ln необходимо, чтобы выполнялось n сумма 2^(-Li)<1 i=1

Теорема Мак-Милана.
Для того чтобы существовал разделимый побуквенный код, необходимо выполнение условия: n сумма 2^(-Li)<=1 i=1
