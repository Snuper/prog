Массивы
Массив - представляет собой фиксированное количество элементов одного типа, все элементы имеют одно имя, но каждый элемент имеет свой номер,обращение к элементам происходит по имени и индексу.

Описание массива Pas: Var имя_массива : array [n1..n2] of тип_элементов; Var a : array [-2..2] of real; C: тип имя[кол-во элементов] int a[5]; Массив - структура с последовательным или случайным доступом к элементам. Часто к элементам массива обращаются последовательно, используя цикл. В памяти элементы массива располагаются последовательно по возрастанию адрессов в памяти. Двумерный массив называют матрицей.

Pas: Var M : array [1..2,1..5] of integer; C: int M[2][3];
