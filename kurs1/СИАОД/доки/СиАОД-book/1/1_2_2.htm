Записи
Записи состоят из фиксированного количества компонентов различного типа. Компоненты называются полями записи.

Pas: TYPE имя_типа = record <поле1> : <тип>; <поле2> : <тип>; ..... end; TYPE zap = record day : byte; month : byte; year : word; end; VAR date : zap; C: struct zap{int day,int month,long year}date; Запись - структура данных со случайным или прямым доступом.
